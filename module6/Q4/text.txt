IP address classes are determined based on the first octet.
Whether an IP is private or public depends on the ranges defined by RFC 1918 for private networks:
Class A Private: 10.0.0.0 to 10.255.255.255
Class B Private: 172.16.0.0 to 172.31.255.255
Class C Private: 192.168.0.0 to 192.168.255.255
